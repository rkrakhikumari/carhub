# ===============================
# SETUP INSTRUCTIONS
# ===============================

# BACKEND SETUP (Python):
# ========================

# 1. Create a folder for backend:
#    mkdir attendance-backend
#    cd attendance-backend

# 2. Create a virtual environment:
#    python -m venv venv
   
#    # On Windows:
#    venv\Scripts\activate
   
#    # On macOS/Linux:
#    source venv/bin/activate

# 3. Install dependencies:
#    pip install -r requirements.txt

# 4. Create app.py file with the Flask code provided

# 5. Run the backend:
#    python app.py
   
#    The API will run on http://localhost:5000

# FRONTEND SETUP (React):
# =======================

# 1. Create a React app (if you don't have one):
#    npx create-react-app attendance-frontend
#    cd attendance-frontend

# 2. Copy Attendance.jsx to src/components/Attendance.jsx

# 3. Update App.js to include:
#    import Attendance from './components/Attendance';
   
#    function App() {
#      return <Attendance />;
#    }
#    export default App;

# 4. Install Tailwind CSS (if not already installed):
#    npm install -D tailwindcss postcss autoprefixer
#    npx tailwindcss init -p

# 5. Configure tailwind.config.js:
#    content: [
#      "./index.html",
#      "./src/**/*.{js,jsx}",
#    ],

# 6. Add to index.css:
#    @tailwind base;
#    @tailwind components;
#    @tailwind utilities;

# 7. Run the frontend:
#    npm start

# USAGE:
# ======

# 1. Start backend first (http://localhost:5000)
# 2. Start frontend (http://localhost:3000)
# 3. Add drivers through API or database
# 4. Mark attendance for drivers
# 5. Export all attendance to Excel with dates as columns

# DATABASE:
# =========

# The SQLite database will be created automatically as attendance.db
